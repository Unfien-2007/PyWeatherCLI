# Current Progress

- Uses OpenWeatherMap to gather weather forecast data.
- Accepts a city name as input before fetching data.
- Displays the current date and time in the terminal when the script runs.
- Shows a Matplotlib graph of 4-day forecast temperature data.
- Adds generated date/time to the forecast graph window.
- Shows a second Matplotlib figure with current weather metrics and timestamp.
- Suggests close city matches from OpenWeatherMap when a city is not found.
- Lets the user retry city input instead of exiting immediately on a missing city.
